# Arquitecto - Premium Architecture Portfolio

A minimalist, highly elegant, and modern architecture portfolio designed for world-class studios. Built purely with HTML5, CSS3, and Vanilla JS. No heavy frameworks or libraries (other than Phosphor Icons for iconography).

## Features
- **Awwwards-style Premium UI/UX:** Large whitespace, cinematic image reveals, typography-driven layout.
- **Micro-interactions:** Custom cursor tracking, hover effects, line reveals, staggering elements.
- **Scroll Animations:** Built-in Intersection Observer based scroll reveals and subtle parallax.
- **Dark Mode:** Seamless light/dark theme toggling with `localStorage` persistence.
- **Responsive:** Fluid typography and spacing, adapts perfectly to all screen sizes.
- **Performance:** No jQuery, no massive libraries. Just vanilla web technologies.

## Structure
- `/css`: Contains modular CSS files (`variables.css`, `style.css`, `responsive.css`, `animations.css`)
- `/js`: Contains modular JS classes (`main.js`, `animations.js`, `scroll.js`, `darkmode.js`, `gallery.js`)
- `*.html`: The multi-page setup for the studio's website.

## Setup
Simply open `index.html` in your modern web browser. For the best experience (especially for local external resource loading like fonts and some JS functionality), run a local development server.

## Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Variables, Flexbox, CSS Grid, Transitions, Keyframes)
- Vanilla JavaScript (Classes, IntersectionObserver, MutationObserver)
- Phosphor Icons (Web Font)
- Google Fonts (Inter, Playfair Display)
